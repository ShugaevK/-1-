# Анализ влияния праздников на выручку магазинов
В данном кейсы были найдены ответы на следующие вопросы:
- Какова средняя недельная выручка каждого магазина в праздничные недели?
- Какова средняя недельная выручка тех же магазинов в обычные недели?
- На сколько процентов праздник повышает выручку по сравнению с обычными неделями для каждого магазина?
- Какие магазины сильнее всего реагируют на праздники и где имеет смысл усиливать маркетинговую активность?
